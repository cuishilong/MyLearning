### 一、逻辑回归

逻辑回归（Logistic Regression）是一种分类模型，属于监督学习范畴，使用的是回归的方法处理分类问题。

根据标签的个数，分类问题大体可分为二分类问题和多分类问题。

逻辑回归是一种常用，简单，高效的分类算法，因而使用广泛。

例如判断用户是否会对帖子点赞，则为二分类问题；判断用户浏览，点赞，收藏，转发等等则为多分类问题。

*注：分类输出离散值，回归输出连续值。*
